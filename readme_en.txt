 1) SYSTEM REQUIREMENTS
________________________

Supported Operating Systems are:

	Windows XP (SP3)
	Windows 7
	Windows 8
	Windows 8.1
	Windows 10

Minimal Hardware Requirements:

	Dualcore processor
	1 GB of RAM
	700 MB of free hard disk space 
	128(+) MB video card	("GeForce FX" or better)
	1 Mbit/s internet connection
	Sound Card
	Keyboard
	Mouse

_______________________________________________________________________

 2) CONFIGURATION
__________________

FOnline 2 available on next languages:

   English
   Russian
   Polish
   Portuguese

For language switching and additional settings use FOConfig (FOConfig**.exe for your language) in game folder. For update run FOUpdater.exe
