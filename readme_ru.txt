 1) СИСТЕМНЫЕ ТРЕБОВАНИЯ
_________________________

Игра поддерживает следующие операционные системы:

	Windows XP (SP3)
	Windows 7
	Windows 8
	Windows 8.1
	Windows 10

Минимальные системные требования:

	Двухядерный процессор
	1 гигабайт оперативной памяти
	700 мегабайт свободного дискового пространства
	видеокарта с 128 мегабайтами памяти ("GeForce FX" или лучше)
	Интернет соединение с пропускной способностью не менее 1 Мбит/с
	Звуковая карта
	Клавиатура
	Мышь

_______________________________________________________________________

 2) НАСТРОЙКИ
______________

FOnline 2 поддерживает следующие языки:

   Русский
   Английский
   Польский
   Португальский

Для смены языка или дополнительных настроек используйте FOConfig (FOConfig**.exe для вашего языка) в папке с игрой. Для обновления используйте FOUpdater.exe.
